Spring 이란
    - 자바의 웹 프레임워크로 자바언어를 기반으로 사용한다
    - 자바로 다양한 어플리케이션을 만들기 위한 프로그래밍 틀이라 할 수 있다.
    - 자바의 활용도가 높아지면서, 자바를 이용한 기술이 JSP, Mybatis, JPA등의 기술이 생겼다
    - Spring은 다른 사람의 코드를 참조하기 쉽고 편리한 구조이다

프레임워크란
    - 프레임워크는 어떠한 목적을 달성하기 위해, 복잡하게 얽혀 있는 문제를 쉽게 해결하기 위한 약속이자 도구이며, 소프트웨어 개발에 하나의 뼈대 역할을 한다.
    - 프레임워크는 자주 쓰일 만한 기능들을 한데 모아 놓은 유틸들의 모음이다.
    - 의자를 만든다고 가정 할때 의자를 만드는 망치 나 못 같은 개념
